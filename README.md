# OOP Practice Project

Yeh chhota C++ project OOP concepts practice ke liye hai.

Files:
- `opp practice .cpp` — Teacher class example (copy constructor, methods)
- `student.cpp` — Student class example
- `course.cpp` — Course class example
- `school.cpp` — Simple School Management demo

Build (Windows, Visual Studio cl.exe):
```powershell
# Example: compile opp practice
cl.exe "opp practice .cpp"
# Or use your IDE's build task
```

Git quick guide (Roman Urdu):

- `git add .` — sari files staging mein dalta hai
- `git commit -m "msg"` — changes save karta hai
- `git remote add origin <URL>` — GitHub repo jodta hai
- `git push -u origin main` — pehli dafa push karta hai
